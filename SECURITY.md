# Security Policy

## Supported Versions

Currently versions in 0.1.x of this project are being supported
with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :white_check_mark: |

## Reporting a Vulnerability

You can report any vulnerability in the issues section or
send an email to [imranmat254@gmail.com](imranmat254@gmail.com).

Expect to get an update on a report vulnerability in 3 days
if the vulnerability is accepted but if the vulnerability is declined then this
means a patch update already exists.
